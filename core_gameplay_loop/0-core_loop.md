Dans Cyberpunk 2077 :

* Le joueur choisit une action → agit (dialoguer, intervenir, explorer) → le monde réagit (réponses, conséquences) → le joueur reçoit un retour (résultat, émotion, impact) → il veut voir les effets de ses prochains choix

Cette boucle fonctionne car chaque action entraîne une réaction qui peut être différente selon le contexte.
* Le joueur est motivé à recommencer pour :

explorer d’autres possibilités
voir d’autres conséquences
améliorer ou changer ses décisions

Boucle simple :
Choisir → agir → observer → ressentir → recommencer
