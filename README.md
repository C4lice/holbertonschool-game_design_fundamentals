## ***AR/VR - Game Design Fundamentals***

___

**Contexte general**
Dans le cadre de ce projet, vous vous concentrerez sur l'expérience du joueur. Avant même les mécanismes, les graphismes ou le code, un jeu est une expérience conçue pour faire ressentir quelque chose au joueur.
Vous apprendrez à identifier et à décrire l'âme d'un jeu : l'émotion, le rêve et le souvenir qu'il laisse au joueur.
___

**Généralités**
